Methods

  • .avatar(user)

    Returns avatar url of specified user. If the user is invalid it will throw an error

    Example:

    console.log(await aops.user.avatar('438336'));

    Output:

    //avatar.artofproblemsolving.com/avatar_438336.jpg?t=1616293663


  • .goals(user)
